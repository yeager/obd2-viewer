# OBD2 Viewer

OBD2 diagnostic data viewer and logger for vehicles.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install obd2-viewer
```

### Fedora/RPM
```bash
sudo dnf install obd2-viewer
```
